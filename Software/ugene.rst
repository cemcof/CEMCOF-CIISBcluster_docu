UGENE
---------

**VERSIONS:**

* 42.0

**GENERAL INFORMATION**

UGENE is free open-source cross-platform bioinformatics software.

**USAGE SPECIFIC INFORMATION**

.. code-block:: console

   $ ssh ciisb.ceitec.muni.cz
   $ module add ugene
   $ ugene --help # help
   $ ugeneui # GUI
