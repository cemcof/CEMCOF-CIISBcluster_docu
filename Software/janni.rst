JANNI
---------

**VERSIONS:**

* 0.2.2

**GENERAL INFORMATION**

Just another noise 2 noise implementation (JANNI) with Keras for cryo-em data. JANNI implements a neural network denoising tool described in NVIDIA's `noise2noise paper <https://arxiv.org/abs/1803.04189>`_.
