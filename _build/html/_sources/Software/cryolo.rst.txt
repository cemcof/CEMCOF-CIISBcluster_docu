.. cryolo:

CRYOLO
------

**VERSIONS:**

* 1.7.6

**GENERAL INFORMATION**

Program for particle picking in single particle electron microscopy data based on convolution neural networks which utilizes the You Only Look Once (YOLO) object detection system. The program can be installed separately or is provided as a part of Sphire package. CrYOLO is developed and maintained by Thorsten Wagner.

**USAGE SPECIFIC INFORMATION**


.. code-block:: console

   module add cryolo

* create either interactive job ``qsub -I`` or prepare the job in the GUI and run it from script

* do not run the program from GUI on the frontend (which has no GPUs)
