.. novactf:

novaCTF
---------

**VERSIONS:**

* 1.0

**GENERAL INFORMATION**

3D-CTF correction program for electron microscopy. The program is maintained and developed by `Beata Turonova`_.

**USAGE SPECIFIC INFORMATION**

* runs on CPU only

.. code-block:: console

   ssh ciisb.ceitec.muni.cz
   module add novaCTF
   novaCTF
