XMIPP
-----

**VERSIONS:**

* 3.20.07

**GENERAL INFORMATION**

A suite of command-line based image processing programs, which are particularly aimed at single-particle 3D electron microscopy. Scipion is developed and maintained by `Jose Maria Carazo lab`_.

**USAGE SPECIFIC INFORMATION**

.. code-block:: console

   ssh -X ciisb.ceitec.muni.cz
   module add xmipp
