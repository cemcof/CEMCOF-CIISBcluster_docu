.. topaz:

TOPAZ
-----

**VERSIONS:**

* 0.2.4

**GENERAL INFORMATION**

Program for particle picking in single particle electron microscopy data using neural networks and positive-unlabeled learning. Topaz is developed and maintained by Tristan Bepler.

**USAGE SPECIFIC INFORMATION**

* loading from the modules

.. code-block:: console

   module add topaz

* it might be more user friendly to run Topaz from :doc:`relion` or :doc:`cryosparc` than directly from command line
