.. frealign:

FREALIGN
---------

**VERSIONS:**

* 9.11 (last version)
* 9.11_1708 (August 2017)

**GENERAL INFORMATION**

Frealign is a of **command-line based programs and scripts** for high-resolution refinement of 3D reconstructions from cryo-EM images of single particles developed in `Nikolaus Grigorieff lab`_.

**USAGE SPECIFIC INFORMATION**

* loading from modules

.. code-block:: console

   ssh ciisb.ceitec.muni.cz
   module add frealign

* the version 9.11_1708 contains **FrealignX** as well
