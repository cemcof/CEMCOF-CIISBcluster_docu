.. unblur:

UNBLUR
------

**VERSIONS:**

* 1.0.2

**GENERAL INFORMATION**

Command-line based **CPU** program for the determination of the motion in electron microscopy movies developed in `Nikolaus Grigorieff lab`_.

**USAGE SPECIFIC INFORMATION**

* loading from modules

.. code-block:: console

   module add unblur
