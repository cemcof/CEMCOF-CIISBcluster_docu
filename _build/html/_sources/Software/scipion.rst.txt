Scipion
-------

**VERSIONS:**

* 3.0

**GENERAL INFORMATION**

Cryo-EM image processing framework integrating different software packages with focus on the traceability of the data analysis workflow. Scipion is developed and maintained by `Jose Maria Carazo lab`_.

**USAGE SPECIFIC INFORMATION**

.. code-block:: console

   ssh -X ciisb.ceitec.muni.cz
   module add scipion
   scipion
