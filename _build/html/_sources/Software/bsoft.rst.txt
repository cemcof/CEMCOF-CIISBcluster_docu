.. bsoft:

BSOFT
---------

**VERSIONS:**

* 1.9.0
* 2.0.7

**GENERAL INFORMATION**

*"Bsoft is a collection of programs and a platform for development of software for image and molecular processing in structural biology."* The package provides workflows for single particle data analysis, electron tomography data analysis, helical reconstruction and other. The package is developed and maintained by `Bernard Heymann`_.

**USAGE SPECIFIC INFORMATION**

* for GUI based applications such as *bshow*

   ssh -X ciisb.ceitec.muni.cz
   module add bsoft
