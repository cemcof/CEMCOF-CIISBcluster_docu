.. motioncor2:

MOTIONCOR2
------

**VERSIONS:**

* 1.4.0

**GENERAL INFORMATION**

Command-line based multi-GPU program for correction of sample and beam-induced motion in cryo-electron microscopy data. The program is developed and maintained by Shawn Zheng at UCSF.

**USAGE SPECIFIC INFORMATION**

.. code-block:: console

   module add motionCor2
