Configuration
-------------

The following computational resources are available:

* single frontend node running Debian 10.10 for submission of the computational tasks and operation of the graphical user interface software

  * accessible at ``ciisb.ceitec.muni.cz``

* ten computational nodes with following configuration:

  * 2 CPUs (AMD Epyc 7302, 32 physical CPU cores, 3.0GHz)
  * 256 GB RAM (DDR4 3200MHz)
  * 4 GPUs (Nvidia Quadro RTX 5000)
  * 10TB drive as a primary scratch
  * 2TB SSD drive as a secondary scratch

* the computational tasks are submitted to PBS system
