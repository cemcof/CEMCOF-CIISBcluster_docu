GENERAL INFORMATION

Comprehensive package for all the steps of the single particle analysis workflow operated through web browser interface. In addition, the cryoSPARC Live enables for the real-time data analysis during data collection. The package is developed and maintained by Structura Biotechnology Inc.

USAGE SPECIFIC INFORMATION

Go to the login page and enter the credentials. When you want to run cryoSPARC Live session, go here. In case you do not yet have an active account, you can send a request for account creation.

  • any new project MUST be saved to the following path upon creation: /storage/brno14-ceitec/shared/cemcof/internal/CIISBcryosparcProjects/

  • use the following binary, when running gctf: Gctf-v1,06_sm_30_cu8,0_x86_64

  • the path to the Topaz executable is: /cemcofsw/em/miniconda3/3.py38_4.9.2/envs/topaz-0.2.4/bin/topaz
